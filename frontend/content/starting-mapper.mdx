# Beginnen met mappen

Iedereen kan bijdragen aan OpenStreetMap. Een goede manier om te
beginnen is om in een gebied te bekijken waar je bekend mee bent. Klop
er iets niet? Pas het aan! Je kan zelf iets aanpassen door op Bewerken
te klikken.

## Maak een account aan

Om wijzigingen te maken in OpenStreetMap heb je een account nodig. Hiervoor heb je alleen een e-mailadres nodig. Je kan je aanmelden via de [OpenStreetMap website](https://www.openstreetmap.org/user/new).

## Kies een editor

Er zijn verschillende editors beschikbaar voor OpenStreetMap. Een simpele editor om mee te beginnen is [iD](https://id.openstreetmap.org/). Dit is een webgebaseerde editor die je direct in je browser kan gebruiken. Een andere populaire editor is [JOSM](https://josm.openstreetmap.de/), een desktop applicatie die meer geavanceerde functies biedt.

## Aanpassen

Binnen de editor kan je verschillende dingen aanpassen. Je kan punten, lijnen en gebieden toevoegen, verwijderen of bewerken.
Om aan te geven wat voor soort object je toevoegt, kan je gebruik maken van de presets. Dit zijn vooraf gedefinieerde instellingen die je helpen om de juiste tags toe te voegen aan je object.
Bijvoorbeeld, als je een restaurant toevoegt, kan je de preset voor restaurant gebruiken.
Deze preset word dan vertaalt naar de juiste tags in OpenStreetMap. Dit maakt het makkelijker om de juiste informatie toe te voegen en zorgt ervoor dat je wijzigingen goed worden weergegeven op de kaart.

## Wiki

Veel van de gebruikte tags zijn te vinden op de [OpenStreetMap wiki](https://wiki.openstreetmap.org/wiki/Main_Page).
Hier kan je meer leren over de verschillende tags en hoe je ze kan gebruiken. Een kantekeing aan de wiki is dat iedereen hem aan kan passen.
Dit kan ervoor zorgen dat de informatie niet representatief ten opzichte van het werkelijke gebruik.

## Alle begin is moeilijk

Wees niet bang om iets fout te doen. De eerste paar edits worden door ervaren mappers gecontroleerd.
Deze kan je dan tips geven om je edits te verbeteren.
Mocht je er niet uitkomen, dan kan je altijd een vraag stellen op het [forum](https://community.openstreetmap.org/c/communities/nl/43) van OpenStreetMap.
