# Download OpenStreetMap

Er zijn verschillende manieren om OpenStreetMap data te downloaden. Welke methode het meest geschikt is, hangt af van je gebruiksdoel. In dit artikel bespreken we een aantal opties.

## Overpass Turbo
[Overpass Turbo](https://overpass-turbo.eu/) is een webgebaseerde tool waarmee je specifieke gegevens uit OpenStreetMap kunt ophalen. Dit is handig als je alleen ge√Ønteresseerd bent in bepaalde objecten of gebieden. Je kunt de Overpass API gebruiken om aangepaste queries te maken en de resultaten in verschillende formaten te exporteren, zoals GeoJSON, KML of GPX.

## OSM XML
Als je geinteresseerd bent in ruwe OSM data van een klein gebied, kun je de [OpenStreetMap website](https://www.openstreetmap.org/) gebruiken om een specifiek gebied te downloaden. De data wordt in OSM XML-formaat aangeboden.

## Gegevens van Geofabrik
[Geofabrik](https://download.geofabrik.de/) biedt een handige manier om OpenStreetMap-gegevens te downloaden per regio. Ze bieden dagelijkse updates van de gegevens en je kunt kiezen uit verschillende formaten, zoals PBF of shapefile. PBF is geschikt om in te laden in een database, terwijl shapefile handig is voor GIS-toepassingen.
